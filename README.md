# Climate Intelligence Dashboard

AI-powered dashboard for global temperature analysis, visualization, and forecasting.

## Features
- Country-wise historical temperature trends
- Interactive year range selection
- Monthly temperature heatmap
- AI-based forecasting using linear regression
- Global temperature ranking
- World map visualization

## Dataset
Source: Berkeley Earth Global Land Temperature Dataset

File used:
GlobalLandTemperaturesByCountry.csv

## How to Run

1. Install Python 3.9+
2. Install packages:

pip install streamlit pandas numpy matplotlib seaborn scikit-learn plotly

3. Run the app:

streamlit run app.py

## Project Purpose
This project helps visualize climate change patterns and predict future temperature trends using machine learning.
